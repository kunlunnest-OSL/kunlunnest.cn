# 📰 昆仑巢新闻维护指南

## 🎯 简单3步，轻松更新网站新闻

### 方法一：使用Excel编辑（推荐✨）

#### 1️⃣ 打开CSV文件
双击打开 `news-data.csv` 文件（用Excel或WPS打开）

#### 2️⃣ 添加/修改新闻
在表格中直接编辑：

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 新闻编号（递增） | 27 |
| category | 分类 | event / media / lab / milestone / cooperation |
| date | 日期 | 2025.07 |
| title | 标题 | EMBA校友联盟参访昆仑巢 |
| desc | 简介 | 在思想碰撞中探寻商业新机遇 |
| image | 图片路径 | /images/event-space-1.jpg |
| tag | 标签（大写） | EVENT / MEDIA / LAUNCH |
| gradient | 渐变色 | from-blue-500 to-cyan-500 |
| link | 新闻链接 | https://mp.weixin.qq.com/... |

**分类说明：**
- `event` - 活动 🎪
- `media` - 媒体报道 📰
- `lab` - 实验室 🧪
- `milestone` - 里程碑 🏆
- `cooperation` - 合作 🤝

**渐变色参考：**
- 蓝色系: `from-blue-500 to-cyan-500`
- 紫色系: `from-purple-500 to-pink-500`
- 橙色系: `from-orange-500 to-red-500`
- 绿色系: `from-green-500 to-emerald-500`

#### 3️⃣ 运行更新
保存CSV文件后，双击运行 `更新新闻.bat`

看到 ✅ 更新成功！就完成了

#### 4️⃣ 刷新网站
在浏览器中刷新网站，即可看到更新

## 📝 操作示例

### 添加一条新闻

1. **打开CSV文件**，滚动到最后一行
2. **复制最后一行**，粘贴到新行
3. **修改内容**：
   ```
   27,media,2025.08,新的媒体报道标题,报道简介,/images/event-space-1.jpg,MEDIA,from-blue-500 to-cyan-500,https://新闻链接
   ```
4. **保存文件**
5. **双击运行** `更新新闻.bat`
6. **刷新网站**

### 修改已有新闻

1. 找到要修改的那一行
2. 直接修改对应的单元格
3. 保存文件
4. 运行 `更新新闻.bat`
5. 刷新网站

### 删除新闻

1. 删除对应的整行
2. 保存文件
3. 运行 `更新新闻.bat`
4. 刷新网站

## 🔧 常见问题

### Q: 运行bat文件提示"未检测到 Node.js"？
**A:** 需要安装 Node.js
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本
3. 安装完成后重启电脑
4. 再次运行 `更新新闻.bat`

### Q: 更新失败，提示"CSV格式错误"？
**A:** 检查以下几点：
- 每行的字段数量是否一致（应该是9个）
- 是否有多余的逗号
- 标题或简介中如果包含逗号，需要用英文双引号包围
- 例如: `"这是标题,包含逗号"`

### Q: 图片如何上传？
**A:** 
1. 把图片放到 `public/images/` 文件夹
2. 在CSV中填写路径，例如: `/images/your-image.jpg`
3. 建议图片命名使用英文

### Q: 如何调整新闻显示顺序？
**A:** 
- 新闻按照CSV中的顺序显示
- 想要调整顺序，直接在Excel中拖动行即可
- 或者修改 `date` 字段（网站按时间倒序显示）

### Q: 想批量导入很多新闻怎么办？
**A:**
1. 准备好Excel表格（按照字段格式）
2. 复制粘贴到 news-data.csv
3. 运行 `更新新闻.bat`

## 💡 小技巧

### 快速复制格式
1. 选中最后一条新闻
2. Ctrl+C 复制
3. 在新行 Ctrl+V 粘贴
4. 只修改内容，保持格式不变

### 使用模板
可以在CSV文件开头保留一行模板（ID设为0），每次复制这行作为起点：
```
0,event,2025.XX,标题模板,简介模板,/images/event-space-1.jpg,EVENT,from-blue-500 to-cyan-500,https://
```

### 批量操作
- Excel的填充功能可以快速生成递增的ID
- 使用查找替换批量修改某个字段
